
# Minitask Manager UI

A simple task management application built with Next.js, TypeScript, TailwindCSS, and React Context API.
Simplify add/update/delete tasks, pagination, dark mode, and user authentication.

## Installation

Install Minitask Manager with npm

```bash
    git clone <repository-url>
    cd minitask-manager
    npm install
```

Make a .env.local file in root project and add this code to hit backend api

```bash
NEXT_PUBLIC_API_URL=backend-public-api
```

Now run locally this project to development in your project

```bash
npm run dev
```
